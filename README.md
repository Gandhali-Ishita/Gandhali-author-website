# Gandhali Ishita Author Website

This is a static website built with plain HTML, CSS, and JavaScript. It can be hosted for free with GitHub Pages.

## Files

- `index.html` — website structure and content
- `styles.css` — design, colors, spacing, and responsive layout
- `script.js` — mobile menu and demo email form behavior

## How to publish on GitHub Pages

1. Go to GitHub and create a new repository.
2. Name it something like `author-website`.
3. Upload these files into the repository:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
4. Click **Commit changes**.
5. Go to **Settings** in the repository.
6. Click **Pages** in the left menu.
7. Under **Build and deployment**, set:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
8. Click **Save**.
9. Wait a minute or two. GitHub will show your live website link.

Your site URL will look like:

`https://your-github-username.github.io/author-website/`

## How to edit the site

Open `index.html` and change the book descriptions, contact email, social links, and author bio.

To add another book, copy one full `<article class="book-card">...</article>` section and edit the title/description.
